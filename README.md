# SimpleMIDISplitChannelModifier
A clean, Python-based simple MIDI split-point channel modifying monitor/router

This file includes:

Four tabs (Routing, Incoming, Outgoing, MIDI Ports)

Split point centered at the top

Lower/Upper zone routing with Unchanged as default

Exact ALSA/MIDO port selection

“No port selected” safe fallback

Incoming/Outgoing filters

Status bar

Thread‑safe routing loop

Ready to run using Python + mido + python‑rtmidi
